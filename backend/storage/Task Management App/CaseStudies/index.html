Use Cases

UC-001: New Task Creation

A user creates a new task in the task management app, including task details such as title, description, and due date.

UC-002: Task Assignment

A user assigns a task to another user in the task management app, including assigning task details such as due date and responsible user.

A user prioritizes a task in the task management app using AI-powered algorithms, including selecting task priority and due date.

UC-004: Task Details

A user views task details in the task management app, including task title, description, and due date.

UC-005: Task Deletion

A user deletes a task in the task management app, including deleting task details such as title and due date.
